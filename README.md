## SamaNetwork for Android

### PREREQUISITES

* JDK 1.8
* Android SDK
  - Android NDK

### BUILD

You can check whether the latest commit builds under UNIX environment by checking Travis status.

* Clone the repo using `git clone https://github.com/SamaNetwork/sm_client_android.git`
* Build it using Android Studio or gradle script
